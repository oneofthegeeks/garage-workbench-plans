Assembly Order

01
Build toe kick bases for all 5 sections. Level them and fasten to the floor and/or wall.
02
Build cabinet boxes — sides, top, bottom, back. Frameless construction using pocket screws and glue.
03
Set boxes on toe kick bases. Screw adjacent boxes together through their shared side panels.
04
Install drawer slides in S1 and S5. Build drawer boxes and test-fit before gluing.
05
Install shelves in S2 and S4. Pocket-screw or use shelf pins — decide before drilling.
06
Laminate workbench top — glue two 3/4" ply layers with construction adhesive, clamp overnight. Offset seams.
07
Attach top to cabinet boxes from below with 2" screws through the cabinet top panels.
08
Attach drawer false fronts using temporary spacers to set a consistent 1/8" reveal on all sides.
09
Place miter saw on S3 platform. Confirm saw table is flush with the 36" bench surface. Shim if needed.
10
Fasten to wall — 3" screws through cabinet backs or a ledger rail into wall studs.

Notes

⚠ Measure Before Cutting S3

The Ryobi saw's table height above its base determines S3's box height. Spec assumes 3". Formula: S3 box height = 30.5" − saw table height. Cutting too short cannot be undone.

MDF & Moisture

MDF swells when wet. Seal all edges and surfaces with paint, shellac, or oil before use. The toe kick bases use plywood for this reason.

Top Seam Placement

Offset the 96"/12" seam between layer 1 and layer 2 by at least 12". Aligned joints weaken the top significantly.

Drawer Slides

Use full-extension ball-bearing slides — Blum, Accuride, or equivalent. Cheap slides fail under tool loads. 22" slides fit a 27" box with clearance.

Shelf Mounting

Choose one method before building: pocket-screw shelves permanently, or drill shelf-pin holes (32mm system) for adjustability. Plan ahead — don't try to do both.
